## Projeto 
O projeto escolhido foi a mesma aplicação desenvolvida para a matéria de Startup One, o Condolives. Esse sistema propõe uma nova abordagem em relação ao modelo tradicional de gestão de condomínios. Oferecendo um aplicativo mobile para uso dos moradores e um painel administrativo para uso dos administradores responsáveis. <br><br>
O Condolives oferece um ambiente completo de acompanhamento e controle para o administrador e uma plataforma útil e amigável para os moradores do condomínio.<br><br>
A solução permite o cadastro de solicitações que são exibidas em formatos de post em um feed, outros moradores podem interagir com esses posts, criando um filtro orgânico que indica as demandas mais urgentes. <br><br>
O aplicativo possibilita a criação de anúncios de venda de produtos, prestação de serviços, doação, necessidade etc. estimulando uma economia interna dos moradores. Também é possível agendar espaços sociais, avisar a portaria sobre visitas, receber notificações de entrega, enxergar de forma transparente os serviços em andamento do condomínio, entre outros. Todos esses dados são tratados, categorizados, relacionados e exibidos ao administrador, fornecendo novas perspectivas com objetivo de tornar as tomadas de decisão mais assertivas e precisas.
